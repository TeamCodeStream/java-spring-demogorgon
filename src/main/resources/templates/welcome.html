After clicking the links above, there will be a message with the transaction/span name that was generated with the request.

There should also be a child span, its name starting with "Custom/" (except for Auto instrumentation only). That span is generated by an instrumentation referenced by the link. These child spans are all manually generated, but may use the same mechanism used by auto instrumentation.
